Acerca o programa

KS Boolean Expression é um software livre usado para minimizar funções bôoleanas por um método gráfico de mapas de Karnaugh.

Útil para os colaboradores de dispositivos digitais pequenos, quanto para àqueles que são familiares com a álgebra bôoleana.

Os colaboradores de KS Boolean Expression quiseram ajudar estudantes da engenharia elétrica assim como o professor da álgebra bôoleana.
