Préférences

Vous pouvez personnaliser votre application en accédant aux préférences. Pour cela aller au menu "Fichier" puis cliquez sur "Préférences", comme le montre la figure suivante:

Vous verrez alors aparaître la boite de dialogue suivante :

Changer la langue:

Pour une manipulation plus facile du logiciel, six(6) langues ont été mises à votre disposition: Anglais, Français, Allemand, Espagnol, Portugais et Italien.
Pour changer la langue, dans "Préférences", sélectionner la langue voulue, puis cliquer sur "Ok".

Changer le thème de l'application:

L'interface des thèmes permet de modifier l'aspect par défaut de KS Boolean Expression.
Pour changer le thème, aller dans "Préférences" puis dans "Apparence", sélectionner le thème voulu.


Modifier le type de solution par défaut:

Par défaut, KS Boolean Expression donne un résultat détaillé de la simplification, c'est-à-dire qu'il affiche en plus de la forme réduite la table de Karnaugh et la table de vérité (Pour des raisons de lisibilité de la solution, cette dernière est affichée uniquement s'il n'y a qu'une fonction à simplifier ).

Vous pouvez changer le type de solution par défaut en allant dans "Préférences", puis dans "Type de Solution" et choisir le type voulu, puis cliquer sur "OK" pour appliquer le changement.
