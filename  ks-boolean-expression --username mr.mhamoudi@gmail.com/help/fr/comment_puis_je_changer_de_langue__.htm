Pour changer la langue de cette application:

" Cliquez sur "Fichier" qui se trouve dans la barre du menu, puis le bouton "Préférences".

" Lorsque vous sélectionnez la langue qui vous convient, cliquez sur "OK" pour appliquer le changement.



Pour plus d'informations, consulter la rubrique "Personnaliser votre application".
