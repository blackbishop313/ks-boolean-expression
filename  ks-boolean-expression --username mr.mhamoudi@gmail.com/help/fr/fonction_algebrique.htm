Nouvelle fonction algébrique

Pour ajouter une nouvelle fonction définie algébriquement, cliquez sur "Ajouter" dans la barre des menus, sélectionnez "Nouvelle fonction", Puis choisir "Forme algébrique".


Vous pouvez aussi l'écrire directement dans champ "Expression" qui se trouve en dessous de la barre d'outils.

Lorsque vous ajoutez une nouvelle fonction algébrique:
• Saisir votre fonction dans le champ formule "Expression".
• Cliquer sur le bouton "Simplifier" pour lancer la simplification de votre fonction, vous verrez directement le résultat s'afficher.

Remarques:

  • L'insertion des opérateurs peut se faire à partir des touches du clavier ou bien directement en utilisant la barre d'outils, il suffit d'un simple clic sur le bouton correspondant.
  • Vous pouvez simplifier plusieurs fonctions simultanément. Pour cela vous n'avez qu'à les introduires dans le champs "Expression", séparées avec des points virgules ";". Faites attention à ce qu'il n'y ait pas d'espace entre les différentes fonctions.

Choisir le type de solution:

Après avoir introduit votre expression, vous pouvez choisir quelle type de solution vous souhaiterai avoir s'afficher sur votre écran. Pour cela il suffit de cliquer sur "Simplifier" dans la barre des menus puis choisir le type de solution:

  • Fonction simplifiée: pour une solution non détaillée (forme réduite uniquement).
  • Fonction détaillée: pour une solution détaillée ( forme réduite + tables).
