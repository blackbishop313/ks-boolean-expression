Table de Karnaugh

Pour ajouter une nouvelle table de Karnaugh, cliquez sur le bouton "Ajouter" dans la barre des menus, sélectionnez "Nouvelle table", puis choisir "Table de Karnaugh".



Avant de remplir la table de Karnaugh, Vérifiez d'abord le nombre de variable que contient votre fonction.

Pour entrer une valeur dans la table de karnaugh, sélectionnez une case en cliquons dessus ,une liste vous apparaîtra puis sélectionnez:

• "1" pour un.
• "0" pour zéro.



Après avoir complété la table, cliquez sur le bouton "OK" pour avoir le résultat de la simplification.
