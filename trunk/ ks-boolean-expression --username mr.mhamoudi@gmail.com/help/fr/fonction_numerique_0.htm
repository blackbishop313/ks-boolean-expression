Sommaire

Nouvelle fonction numérique:

Pour ajouter une nouvelle fonction numérique, cliquez sur le bouton "Ajouter" dans la barre des menus, sélectionnez "Nouvelle fonction", puis choisir "Forme numérique".

Par exemple si vous disposer de la fonction suivante: F (A, B, C, D) =∑ (2, 7, 10, 15).

les valeurs 2, 7, 10, 15 sont les index des lignes de la table de vérité qui sont à un (1), les index qui restent seront mis à zéro (0).
Fixer le nombre de variable on cliquant sur le bouton "Nombre de variables" (ici on à quatre variables).

Lors de l'insertion des index, le remplissage d'un seul champ (UN ou ZERO) est suffisant, puisque le champ restant va se remplir automatiquement par les valeurs restantes.

Entrer 2, 7, 10,15 dans le champ "1". Cliquez sur "OK".

Remarques:

  • Pour séparé entre les index, on utilise la virgule (,).
  • Lors de l'insertion des adresses veuillez respecter les index autorisés.
